# Workshop Management System

## Description
Workshop Management System is a Flutter mobile application designed to help manage vehicle repair workshop operations efficiently.
It allows workshop owners to manage inventory, schedules, and employee payroll in one platform. Foremen can view and select available
work schedules and check their payment records.The system also supports inventory requests to prevent stock shortages.
Additionally, it includes a rating feature to collect feedback and improve workshop service quality.

## Technologies
- Framework: Flutter (Dart SDK ^3.8.0)
- Backend / services: Firebase (Authentication + Cloud Firestore)
- Packages: firebase_core, firebase_auth, cloud_firestore, intl, flutter_lints

## How to Login
- U can register an account first and login with password to the application
